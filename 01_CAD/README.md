# CAD

This folder contains all CAD models developed for the Stair Climbing Wheelchair project.

## Contents

- Part Models (.SLDPRT)
- Assembly Models (.SLDASM)
- STEP Files (.STEP)
- STL Files (.STL)
- CAD Configurations
- Design Iterations

## Software Used

- SolidWorks 2026

The CAD files represent the complete mechanical design of the stair-climbing wheelchair, including individual components and final assembly.
