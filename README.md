# TerraHopper: Affordable Stair-Climbing Wheelchair

## Project Overview

This project presents the design and development of **TerraHopper**, an affordable stair-climbing wheelchair designed to improve mobility for elderly and differently-abled individuals. The solution integrates an innovative sun-planet gear transmission, lever-operated mode switching, ergonomic design principles, and a hydraulic disc braking system to enable safe and efficient movement on both flat surfaces and stairs.

---

## Objectives

- Design an affordable stair-climbing wheelchair
- Improve mobility and accessibility for elderly and differently-abled users
- Develop an innovative mechanical power transmission system
- Ensure user safety through an integrated braking mechanism
- Promote sustainable and locally manufacturable product design
